# arviz-meta

ArviZ metapackage for individual ArviZ packages.

## Installation

We use [uv][1] to manage the ArviZ workspace that contains the individual packages; [arviz-base][2],
[arviz-stats][3], and [arviz-plots][4]. You will need to install `uv` before being able to
contribute to the ArviZ metapackage repo. Follow the instructions in the `uv` documentation to
install it on your specific platform.

[1]: <https://docs.astral.sh/uv> "uv"
[2]: <https://github.com/arviz-devs/arviz-base> "ArviZ Base"
[3]: <https://github.com/arviz-devs/arviz-stats> "ArviZ Stats"
[4]: <https://github.com/arviz-devs/arviz-plots> "ArviZ Plots"
